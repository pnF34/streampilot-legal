## 🇫🇷 Français

# Politique de confidentialité de StreamPilot

## 1. Introduction

Cette politique de confidentialité explique quelles données StreamPilot peut collecter, pourquoi elles sont utilisées et comment elles sont protégées.

StreamPilot est un bot Discord conçu pour aider les streamers à configurer, organiser et automatiser leur serveur Discord.

En utilisant StreamPilot, tu acceptes cette politique de confidentialité.

## 2. Données collectées

StreamPilot peut enregistrer uniquement les données nécessaires au fonctionnement du bot.

Ces données peuvent inclure :

- l’ID du serveur Discord
- les IDs des salons configurés
- les IDs des rôles configurés
- le nom de chaîne Twitch configuré
- les liens de réseaux sociaux ajoutés
- le statut du serveur : Starter ou Pro
- les paramètres de bienvenue
- les paramètres de logs
- les données nécessaires aux sauvegardes serveur
- certaines informations techniques liées aux automatisations

StreamPilot ne collecte pas volontairement les messages privés des utilisateurs.

## 3. Utilisation des données

Les données sont utilisées pour permettre au bot de fonctionner correctement, notamment pour :

- créer ou réparer la structure du serveur
- afficher la configuration du serveur
- gérer le lien Twitch
- afficher les réseaux sociaux
- envoyer des messages de bienvenue
- gérer les logs configurés
- gérer les fonctionnalités Starter et Pro
- effectuer des sauvegardes et restaurations lorsque la fonction est disponible
- faire fonctionner les automatisations

## 4. Données Discord

StreamPilot fonctionne via l’API Discord.

Certaines informations comme les IDs de serveur, salons, rôles et utilisateurs peuvent être nécessaires pour exécuter les commandes du bot.

Ces IDs servent uniquement au fonctionnement technique du bot.

## 5. Données Twitch et réseaux sociaux

Si un administrateur configure une chaîne Twitch ou des liens sociaux, StreamPilot peut enregistrer ces informations afin de les afficher avec les commandes du bot.

Cela peut inclure :

- nom de chaîne Twitch
- lien TikTok
- lien YouTube
- lien Instagram
- autres liens sociaux ajoutés via la configuration

## 6. Sauvegardes serveur

Les fonctionnalités de sauvegarde peuvent enregistrer certaines informations de configuration du serveur, comme les salons, rôles, catégories et permissions.

Ces données servent uniquement aux fonctions de backup et restauration de StreamPilot Pro.

## 7. Partage des données

StreamPilot ne vend pas les données utilisateurs.

Les données ne sont pas partagées avec des tiers à des fins publicitaires.

Certaines données peuvent transiter par Discord ou Twitch, car le bot utilise leurs API respectives pour fonctionner.

## 8. Conservation des données

Les données sont conservées tant que StreamPilot est utilisé sur le serveur ou tant qu’elles sont nécessaires au fonctionnement des fonctionnalités configurées.

Si le bot est retiré d’un serveur, certaines données peuvent rester stockées temporairement pour permettre une restauration ou une demande de support.

Une suppression peut être demandée via le serveur support.

## 9. Suppression des données

Le propriétaire ou un administrateur du serveur peut demander la suppression des données associées à son serveur.

Pour demander une suppression, contacte le support officiel :

https://discord.gg/NrQzkttyS2

Merci d’indiquer l’ID du serveur concerné afin que la demande puisse être traitée.

## 10. Sécurité

Nous faisons notre possible pour limiter les données stockées et protéger la configuration du bot.

Cependant, StreamPilot est actuellement en bêta publique. Des bugs ou problèmes techniques peuvent arriver.

Les tokens, clés API et informations sensibles du bot ne doivent jamais être partagés publiquement.

## 11. Responsabilité des administrateurs

Les administrateurs de serveur sont responsables des informations qu’ils ajoutent dans StreamPilot, comme les liens sociaux, les chaînes Twitch ou les paramètres de configuration.

Il est recommandé de vérifier régulièrement les permissions accordées au bot et les données configurées.

## 12. Modifications de cette politique

Cette politique peut être mise à jour avec l’évolution de StreamPilot.

Continuer à utiliser StreamPilot après une mise à jour signifie que tu acceptes la nouvelle politique.

## 13. Contact

Pour toute question concernant la confidentialité ou les données, rejoins le serveur support officiel :

https://discord.gg/NrQzkttyS2

## 🇬🇧 English

# StreamPilot Privacy Policy

## 1. Introduction

This Privacy Policy explains what data StreamPilot may collect, why it is used and how it is protected.

StreamPilot is a Discord bot designed to help streamers configure, organize and automate their Discord server.

By using StreamPilot, you agree to this Privacy Policy.

## 2. Data collected

StreamPilot may store only the data required for the bot to work.

This data may include:

- the Discord server ID
- configured channel IDs
- configured role IDs
- the configured Twitch channel name
- added social media links
- the server status: Starter or Pro
- welcome settings
- log settings
- data required for server backups
- certain technical information related to automations

StreamPilot does not intentionally collect users’ private messages.

## 3. Use of data

Data is used to allow the bot to work properly, including to:

- create or repair the server structure
- display the server configuration
- manage the Twitch link
- display social links
- send welcome messages
- manage configured logs
- manage Starter and Pro features
- perform backups and restores when the feature is available
- run automations

## 4. Discord data

StreamPilot works through the Discord API.

Some information such as server IDs, channel IDs, role IDs and user IDs may be required to execute the bot’s commands.

These IDs are used only for the bot’s technical operation.

## 5. Twitch and social media data

If an administrator configures a Twitch channel or social links, StreamPilot may store this information in order to display it with the bot’s commands.

This may include:

- Twitch channel name
- TikTok link
- YouTube link
- Instagram link
- other social links added through the configuration

## 6. Server backups

Backup features may store certain server configuration information, such as channels, roles, categories and permissions.

This data is used only for StreamPilot Pro backup and restore features.

## 7. Data sharing

StreamPilot does not sell user data.

Data is not shared with third parties for advertising purposes.

Some data may pass through Discord or Twitch because the bot uses their respective APIs to work.

## 8. Data retention

Data is kept as long as StreamPilot is used on the server or as long as it is required for the configured features to work.

If the bot is removed from a server, some data may remain temporarily stored to allow restoration or support requests.

Data deletion may be requested through the support server.

## 9. Data deletion

The server owner or an administrator may request deletion of data associated with their server.

To request deletion, contact the official support server:

https://discord.gg/NrQzkttyS2

Please include the relevant server ID so the request can be processed.

## 10. Security

We do our best to limit stored data and protect the bot configuration.

However, StreamPilot is currently in public beta. Bugs or technical issues may occur.

Tokens, API keys and sensitive bot information must never be shared publicly.

## 11. Administrator responsibility

Server administrators are responsible for the information they add to StreamPilot, such as social links, Twitch channels or configuration settings.

It is recommended to regularly review the permissions granted to the bot and the configured data.

## 12. Changes to this policy

This policy may be updated as StreamPilot evolves.

Continuing to use StreamPilot after an update means that you accept the new policy.

## 13. Contact

For any privacy or data-related question, join the official support server:

https://discord.gg/NrQzkttyS2
